# Cybersecurity-domains

Here’s a quick breakdown of those cybersecurity domains so you understand what each one involves and how they differ:

---

## 🧮 Cryptography

Cryptography is the science of securing information by transforming it into unreadable formats and then back into readable formats using keys.

### What it covers

* Encryption & decryption
* Hashing
* Digital signatures
* Public Key Infrastructure (PKI)
* Secure protocols (TLS/SSL)

### Tools

* OpenSSL
* Hashcat
* John the Ripper
* GPG

### Use cases

* Secure communications
* Password storage
* Data integrity
* Digital transactions

---

## 🕵️ OSINT (Open Source Intelligence)

OSINT is collecting information from publicly available sources to gather intelligence.

### What it covers

* Social media analysis
* Domain & DNS records
* Public records
* Metadata analysis
* Geolocation clues

### Tools

* Maltego
* Shodan
* theHarvester
* Google Dorks

### Use cases

* Reconnaissance in penetration testing
* Threat intelligence
* Investigations

---

## 🗂️ Forensics

Digital forensics involves investigating digital devices and data to uncover evidence.

### What it covers

* Disk forensics
* Memory forensics
* Network forensics
* File recovery
* Evidence preservation

### Tools

* Autopsy
* FTK Imager
* Volatility
* Wireshark

### Use cases

* Incident investigations
* Cybercrime evidence
* Malware analysis support

---

## 🦠 Malware Analysis

Malware analysis is studying malicious software to understand its behavior and impact.

### What it covers

* Static analysis
* Dynamic analysis
* Reverse engineering
* Behavior analysis

### Tools

* IDA Pro
* Ghidra
* Cuckoo Sandbox
* Radare2

### Use cases

* Threat detection
* Creating signatures
* Understanding attack techniques

---

## ⚔️ Offensive Security

Offensive security focuses on ethical hacking and identifying security weaknesses before attackers do.

### What it covers

* Penetration testing
* Exploitation
* Vulnerability assessment
* Social engineering

### Tools

* Metasploit
* Nmap
* Burp Suite
* Wireshark

### Use cases

* Security testing
* Red teaming
* Bug bounty hunting

---

## How they relate

* **OSINT** → Reconnaissance
* **Cryptography** → Data protection
* **Forensics** → Post-incident investigation
* **Malware Analysis** → Understanding threats
* **Offensive Security** → Finding weaknesses before attackers
